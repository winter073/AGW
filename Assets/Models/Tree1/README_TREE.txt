HOW TO APPLY TREE MATERIALS



TREE BARK :


1. Create Material for the Tree Bark

2. Set the Tree Bark Material to a AUTODESK INTERACTIVE SHADER

3. Apply all textures to the Tree Bark Material



LEAVES:

1. Create Material for the Tree Leaves

2. Set the Tree Leaves Material to a CICONIA SHADER

3. Apply Bark LeafsCOMBINED_Texture file to 'BASE COLOR' (Located at the very top)

4. Apply Bark LeafsCOMBINED_Alpha file to 'CUTOUT MASK' (Located at the very bottom)

5. Set Mask Clip Value to '0.8'(Located at the very top)

6. Set Height Scale to 0 (Located in the middle)
